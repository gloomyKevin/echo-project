# 产品头脑风暴

## 一句话描述

帮你发现生活的可能性，推动你去体验，在每一次发生中认识自己。

## 出发点

AI 革命后，世界越来越丰富，生活应该越来越充实而不是越来越贫瘠。人应该更多建立和世界的连接，更多跳出舒适区——而不是被包裹在信息茧房变得极端，不是完全沉沦于线上，不是自叹于世界的割裂。

## 核心理念

**认识自己 → 发现可能 → 发生即奇迹**

先帮你看清自己是谁，再帮你看见生活还能怎么过，然后推你一把去体验——每一次发生都是奇迹。

未来拓展方向：找到同好、共同分享（不是现阶段重点）

## 产品设计原则

- **Don't make me think** —— 懒人式、选择式为主，不要让用户思考
- **如无必要，勿增实体** —— 克制，不堆功能
- **Social Proof（社会认同）** —— 让用户看到别人的选择和体验，降低决策门槛

---

## 愿景分层

### 第一阶段：发现与体验（当前聚焦）
- 做一个交互创新的生活可能性发现工具
- 通过卡片池让用户看到"原来我还可以这样"
- 核心价值：不是替你做决定，而是帮你发现你不知道自己想要的东西
- 通过每一次交互，自然沉淀用户的偏好、价值观、性格数据

### 第二阶段：人格数据化引擎（下一步）
- 积累足够数据后，训练/微调模型
- 构建用户的「数字人格档案」——性格、偏好、价值排序、决策模式
- 从"帮你发现可能"进化到"真正懂你的 agent"

### 第三阶段：通用人格规范（终极野心）
- 把「数字人格档案」做成开放规范/协议
- 任何 AI 产品都能一键导入这份存档
- 用户换一个 AI 不用从零开始，所有 AI 都能秒懂你
- 你的人格数据属于你自己，可迁移、可授权、可撤回

## 原子行为体系

不管是代码实现、模型训练还是自然语言到选择的映射，都需要一套自建的、可以替代人的自然行为或意识的原子体系。

### 原子维度（举例）
- **时间**：不加班的周末、清明假期、说走就走……
- **财务**：这个月月光了、有闲钱、预算有限……
- **事件**：看电影、旅游、拜访朋友、买东西……
- **舒适区突破**：恐怖片、没去过的城市、没试过的食物……
- **导向结果**：能量恢复、社交感情增加、视野拓展……
- **外部状态**：事事不顺、信命、想改运势……

### 原子组合示例
- 周末 + 不加班 + 月光 → 看电影 → 挑战恐怖片 → 看完找朋友聊 → 能量恢复 + 社交↑
- 清明假期 + 不考试 + 说走就走 + 爱好旅游 → 去清迈
- 事事不顺 + 信命 + 改运势 → 换城市 → 城市选择卡片

### 技术策略
- 前期没有 token 资源时，先做离线枚举和预设
- AI 离线缓存一批原子数据
- 后续逐步接入 AI 实时生成

## 卡片系统（核心交互）

### 卡片类型与层级

**从微粒卡到主线卡的梯度：**
- **微粒卡**："今天上床不带手机"、"买一杯没试过味道的奶茶"
- **日常卡**：看一部电影、去一家新餐厅
- **事件卡**：周末露营、短途旅行
- **大爆炸卡**：如"不要错过春天"——包含多个子卡片项，AI 给出连贯的事件安排建议
- **围观卡**：有趣的尝试，如"高中生特种兵游玩新加坡"——选择尝试后，AI 基于你的现状做实现路径拆解
- **抽象里程碑卡**：高考、考研、毕业等人生大节点——独立路径 + 点亮 + 彼此留言鼓励

### 卡片内部机制
- **丰富度分级**：内容覆盖广度
- **置信度分级**：推荐把握程度
- **可执行度分级**：基于用户当前状态是否真的能做
- **红线机制**：不能推荐的边界
- **层层筛选**：从生成到呈现的过滤管道
- 前期尽可能丰富，讨好用户

### 卡片交互创意
- 首页呈现形式待深挖（信息流？卡片墙？）
- 左右滑选择
- 跳出舒适区类型 → 投骰子/随机选择，降低心理门槛
- 卡片翻转可看热度、留言、鼓励
- 前期最小锚点："今天上床不带手机"、"买一杯没试过的奶茶"这类微行动

### 选择的预演
在选择做出之前，AI 基于对世界/经验的了解，给出：
- 过程预演：中间会经历什么
- 结果预演：可预期的结果
- 指标变化：六维度量衡的预期变动

### 完成后的体验
- 完成卡片后，选择极简贴纸表达感受：治愈、开心、释然、疲惫、突破、平淡
- 生成专属编号的碎片卡 + 微光特效
- 点亮时触发轻微震动 + 微光动画
- **每一次行动都有专属记忆点**

## 六维度量衡

所有数据基本收束在六个指标中：

| 维度 | 含义 |
|------|------|
| 健康 | 身体与心理状态 |
| 财务 | 经济状况与消费 |
| 事业 | 工作、学业、成长 |
| 社交 | 人际关系与连接 |
| 环境 | 生活环境与物理世界 |
| 可能性 | 视野拓展、舒适区突破 |

### 数据采集方式
- **绝不做 MBTI 等问卷测试**
- 通过前期自然的选择、长期的沉淀，自动默默得出对你的了解
- 每次卡片交互、选择、完成、反馈都在自然采集

### 第一个里程碑：人生蓝图
数据积累到一定量后，生成你的人生规划蓝图说明：
- 基础情况
- 日常拆解
- 社交圈
- 爱好与讨厌
- 尝试过的努力
- 当下的焦虑
- 短期目标、长期目标

生成后可精细调整，定下人生里程碑后，自动将大需求拆解为小事情，陪伴、提醒、不断给正反馈。

## 反馈追踪与智能回调

### 反馈闭环
发现 → 体验 → 反馈 → 更懂你 → 推荐更准

### 不发声者的指引
- 软件会温和提醒你记录美好生活
- 极简反馈方式：克制的点赞、光亮叠加、贴纸
- 记录本身就是奖励

### 智能回调（数据积累后）
基于历史行为的跨场景关联推荐：
- 之前在家看过吉卜力电影 → 这次去唱片店提醒你可以买一张吉卜力的碟
- 上个月尝试了露营 → 这个月推荐相关的户外活动

---

## 与现实世界的连接：点亮

### 热力地图
- 卡片打卡后，在真实地图上点亮足迹
- 基于所有用户数据形成热力地图
- 看见别人在哪里体验了什么
- 是判断"这个 task 好不好、值不值得、多少人做过"的依据

### 抽象里程碑点亮
- 高考、考研等人生大节点做成抽象路径路线
- 配合点亮 + 留言互相鼓励
- 可以有专版爆炸卡或独立路径

---

## 弱社交

### 社区共建卡片
- 用户输入自然语言，AI 语义转化为标准卡片格式
- 前期没 token 时，开放用户按模板手工创建
- 共创卡片需灰度过滤，筛选出优质内容

### 弱连接体现
1. **卡片热度**：颜色深浅表示热度，翻转/交互可看大家的留言和鼓励
2. **地图点亮**：大家汇聚留下的一个个亮点，直观的社会认同
