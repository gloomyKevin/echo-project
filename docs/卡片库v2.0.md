# 卡片库 v2.0

> **选卡原则**：每张卡必须通过这三个问题——
> 1. 预演文字有没有说出"做这件事的人不会提前知道但做了才懂"的东西？
> 2. 标题够不够具体，不是"多运动"那种建议，而是"今晚上床不带手机"那种行动？
> 3. 做完之后有没有值得记住的感受？
>
> **调性**：温暖、不催促、有点文学性。不是 to-do list，是邀请。

---

## 字段说明

每张卡包含以下关键字段，供推荐算法使用：

**`intensity`（强度）**：推荐算法的核心调节变量。
- `低`：几乎零门槛，随时可以开始，不需要克服任何心理阻力
- `中`：需要一点点计划或克服轻微惯性
- `高`：需要勇气、较多时间投入，或跨出明显舒适区

强度影响推荐策略：新用户先推低强度建立信任 → 随打卡积累逐步引入中/高强度。

**聚类维度**：每张卡的 `atoms.context`（solo/social/indoor/outdoor）+ `domain` 共同决定聚类信号。
推荐系统用这些字段识别用户偏向，并保证推荐池在各维度都有覆盖。

## 聚类覆盖矩阵

确保推荐池不会在某个聚类方向上断货。当前总计 **42 张**。

| 聚类方向 | 代表卡片 | 数量 |
|---------|---------|------|
| 独处·内省·室内 | 不带手机、写三件好事、躺着不做事 | 10 |
| 独处·探索·户外 | 换路回家、地铁到终点、爬山不戴耳机 | 9 |
| 社交·连接 | 给久未联系的人发消息、约只见过几面的人吃饭 | 8 |
| 创意·感官 | 听完整张专辑、去书店随便翻、今晚只点蜡烛 | 7 |
| 身体·挑战 | 骑车去平时打车的地方、报名陌生领域课程 | 5 |
| 高强度·突破 | 做一件圈子里没人做过的事、做一直回避的第一步 | 3 |

**前 8 张展示顺序**（冷启动信号效率最大化）：
```
1. 独处·内省（低强度）  → 测试：内省倾向
2. 独处·探索（低强度）  → 测试：探索倾向
3. 社交·连接（低强度）  → 测试：社交倾向
4. 身体·户外（低强度）  → 测试：行动倾向
5. 创意·室内（低强度）  → 测试：创意倾向
6. 内省·中强度          → 测试：深度参与意愿
7. 探索·中强度          → 测试：城市行动力
8. 社交·中强度          → 测试：社交主动性
```
5 次滑动后即可初步软聚类，8 次后结果可信。

### 今晚上床不带手机

**副标**：把手机放在够不到的地方，然后躺下来。

**时间**：0 分钟（决定即完成）　**预算**：¥0　**场合**：家
**域**：内省　**情境**：独处　**触发心情**：疲惫 / 焦虑

**预演**：
头十分钟可能有点难受，会想找它。不是焦虑，就是手有点空。第二天早上醒来会有一种很轻的感觉——不一定是因为睡得更好，而是因为醒来的第一件事是自己的意识，不是通知。

**预演摘要**：头十分钟手有点空。第二天醒来，第一件事是自己的意识，不是通知。

**维度影响**：健康 +5~12
**标签**：睡眠 · 内省 · 随时可做

### 给一个很久没联系的人发条消息

**副标**：不用说什么大事，就是「最近怎样」。

**时间**：2 分钟　**预算**：¥0　**场合**：任何地方
**域**：社交　**情境**：独处/社交皆可　**触发心情**：孤独 / 心情好 / 想到某人

**预演**：
发出去的瞬间会有点奇怪，像是打扰了什么。但几乎所有人收到都会很开心，因为他们也在想你，只是也没发出来。大概率会聊几句，偶尔聊很久。那种久违的熟悉感，比你预计的要强。

**预演摘要**：发出去的瞬间有点奇怪。但几乎所有人收到都很开心，因为他们也在想你。

**维度影响**：社交 +6~15
**标签**：社交 · 随时可做 · 零成本

### 写下今天三件还不错的小事

**副标**：不用很大，咖啡味道好、座位靠窗都算。

**时间**：3 分钟　**预算**：¥0　**场合**：任何地方
**域**：内省　**情境**：独处　**触发心情**：疲惫 / 焦虑 / 睡前

**预演**：
有时候很难想，有时候想了停不下来。刚开始会觉得有点刻意。做过几次以后会发现自己白天就开始留意这些事了——这才是它真正在做的事，不是晚上的记录，是白天的注意力。

**预演摘要**：刚开始会觉得刻意。做过几次后，你开始在白天就留意这些事。这才是它真正的作用。

**维度影响**：健康 +5~10
**标签**：内省 · 睡前 · 随时可做

### 吃饭时不刷手机，只是吃

### 关灯躺着什么都不做十分钟

**副标**：不刷手机，就让脑子随便跑。

**时间**：10 分钟　**预算**：¥0　**场合**：家
**域**：内省　**情境**：独处　**触发心情**：疲惫 / 焦虑 / 睡前

**预演**：
刚开始会觉得无聊，想找点什么事做。五分钟之后脑子开始自己走动——想到明天，想到很久前的一件事，想到一个莫名其妙的问题。研究把这叫做 mind wandering，说它对整合记忆和产生新想法很重要。大多数人因为手机已经很久没给过脑子这个机会了。

**预演摘要**：五分钟后脑子开始自己走动。大多数人因为手机，已经很久没给过它这个机会了。

**维度影响**：健康 +4~9
**标签**：内省 · 睡前 · 随时可做

### 随机找一张你不熟悉的专辑，听完整张

**副标**：不快进，不换歌。

**时间**：40~60 分钟（可以同时做其他事）　**预算**：¥0　**场合**：任何地方
**域**：艺术　**情境**：独处　**触发心情**：无聊 / 在家 / 想放松

**预演**：
头几首可能没感觉。通常第三或第四首会有一首意外击中你的。听完整张，你会对这个人有一种完整但说不清楚的印象——不只是几首单曲，而是一种氛围。这和刷歌单完全是两件事。

**预演摘要**：头几首可能没感觉。但听完整张，你会对这个人有种完整但说不清楚的印象。和刷歌单完全不同。

**维度影响**：可能性 +4~9
**标签**：艺术 · 音乐 · 随时可做

### 换一条路回家

**副标**：不走今天来时的路。

**时间**：+5~15 分钟　**预算**：¥0　**场合**：附近
**域**：探索　**情境**：独处　**触发心情**：疲惫 / 无聊

**预演**：
你会路过一些从没注意过的东西。一家小店、一棵树、一段台阶、一条不知道通向哪里的小路。不一定有什么特别的，但你会想「原来这里是这样」。熟悉的城市里有一小块，变得不那么熟悉了。

**预演摘要**：你会路过一些从没注意过的东西。不一定有什么，但你会想「原来这里是这样」。

**维度影响**：环境 +4~8 · 可能性 +2~5

**预演摘要**：把决定权交出去是奇怪的事。更有意思的是，不盯菜单的时候，你才真的在这里。

**维度影响**：可能性 +5~10 · 健康 +2~5

**预演摘要**：出了地铁，没有任何计划，只有你和那个地方。这种没有目的地的出发感，比任何精心安排的行程都让人记得住。

## 设计备注

### 为什么是 18 张？
冷启动够用两到三天，不多也不少。用户刷完一轮后，应该已经有 2~3 张加入了队列，有 1 张做了打卡——这时候产品的核心价值已经被感知到了。

### 下一批扩展方向
优先补充的几个方向：
- **雨天专属**：有一类卡片只有下雨天才合适（在咖啡馆等雨停、雨中步行），目前没有覆盖

**时间**：一整天　**预算**：¥0　**场合**：任何地方
**域**：内省 · 健康　**情境**：独处　**强度**：中
**触发心情**：疲惫 / 焦虑 / 想清醒一下

**预演**：
前两个小时会很频繁地拿起手机再放下。下午开始，那种「想看看有没有新东西」的冲动会变少。晚上你会发现比平时多做了几件事，因为你没有了「顺手刷一下」的出口。第二天早上会有一种很清醒的感觉。

**预演摘要**：下午开始冲动会变少。晚上你会发现多做了几件事——因为没了「顺手刷一下」的出口。

**维度影响**：健康 +8~15　**标签**：内省 · 数字戒断

#### 去附近一家菜市场逛一圈

**副标**：不用买什么，就是逛。

**时间**：45~60 分钟　**预算**：¥0~50　**场合**：附近
**域**：探索 · 环境　**情境**：独处　**强度**：低
**触发心情**：无聊 / 周末 / 想出门但不想走远

**预演**：
菜市场里有一种超市里没有的东西——人和食材之间没有包装隔着。你会看到超市里看不到的品种，听到别的地方听不到的对话。逛完出来，对这个城市里的日常会多了解一点点什么。

**预演摘要**：人和食材之间没有包装隔着。逛完出来，对这个城市的日常会多了解一点点什么。

**维度影响**：环境 +5~10 · 可能性 +3~6　**标签**：探索 · 附近 · 社区

#### 去找到离你最近的、你从没去过的公园，在那里坐够三十分钟

**副标**：不用做什么，就是坐着。

**时间**：60~90 分钟（含往返）　**预算**：¥0　**场合**：附近
**域**：环境 · 内省　**情境**：独处　**强度**：低
**触发心情**：疲惫 / 焦虑 / 无聊

**预演**：
找到那个公园的过程有时候比待在里面更有意思——原来这里有个公园。坐够三十分钟比你预期的容易，因为户外的感知节奏和室内不一样。离开的时候，那个地方就从「不知道有」变成了「我的地图里的一个点」。

**预演摘要**：坐够三十分钟比你预期的容易，户外的感知节奏和室内不一样。离开时，那个地方变成了你地图里的一个点。

**维度影响**：健康 +5~10 · 环境 +6~12　**标签**：自然 · 附近 · 随时可做
